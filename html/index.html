News/Patch Log


Patch notes: Version 1.3.0

Date: 27/04/2024

This update brings Solana to the wallet! Now you can easily store and send Solana with this wallet.

Note that you will not be able to send exactly the amount to set, since Solana charges wierd network fees.

Be sure to reimport your wallet to get access to Solana!

Author: DcruBro @ https://dcrubro.com/


Patch notes: Version 1.2.0

Date: 31/03/2024

This update brings Bitcoin to the wallet! Now you can easily store and send Bitcoin with this wallet.

Note that transactions may take a long time, since HardWallex is not made for constant sending and receiving, and as such, uses smaller fees.

You now also have a QR Code for each wallet that you can easily scan to send crypto to your wallet.

VERY IMPORTANT NOTE: You will not see a BTC wallet if you updated your wallet. You will need to re-import your wallet by going to "Settings" and clicking "Re-Import Wallet Data". This will regenerate all wallet data using your mnemonic phrase and will be required after every update until an update system is introduced.

Author: DcruBro @ https://dcrubro.com/


Patch notes: Version 1.1.0

Date: 25/03/2024

This update brings a new feature: Wallet Importing. Previously you could only create a brand new wallet, and if you wanted to import an existing wallet you had to manually replace files and it was an all around hassle.

Now, when missing a wallet, you can simply import an existing wallet using the mnemonic phrase of the original wallet.

Author: DcruBro @ https://dcrubro.com/
